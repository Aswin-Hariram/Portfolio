## 🔧 Installation & Setup

1. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```
2. **Install dependencies**  
   ```bash
   npm install
   ```
3. **Run the development server**  
   ```bash
   npm run dev
   ```
